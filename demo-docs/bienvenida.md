# Bienvenido a MarkApp

Esta es tu primera página de ejemplo. MarkApp te permite organizar tus notas en markdown con una interfaz estilo Notion.

## Características

- Árbol de navegación lateral
- Modo vista y edición
- Crear páginas en carpetas y subcarpetas
- Todo guardado en tu sistema de archivos local

## Prueba esto

1. Click en el botón **+** para crear una nueva página
2. Selecciona una carpeta existente o crea una nueva
3. Cambia entre modo **Vista** y **Edición**
4. Guarda con `Ctrl+S` o `Cmd+S`

## Formato Markdown

Puedes usar todo el poder de markdown:

**Negrita**, *cursiva*, `código inline`

```javascript
// Bloques de código
console.log("Hello MarkApp!");
```
