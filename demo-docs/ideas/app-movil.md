# Idea: App Móvil

Concepto para una aplicación móvil innovadora.

## Concepto

Una app que ayude a las personas a organizar sus ideas usando markdown en el móvil.

## Características principales

1. **Sincronización offline-first**
   - Funciona sin conexión
   - Sync automático cuando hay red

2. **Markdown nativo**
   - Editor optimizado para móvil
   - Preview en tiempo real

3. **Organización**
   - Carpetas ilimitadas
   - Tags y búsqueda
   - Enlaces entre notas

## Mercado objetivo

- Desarrolladores
- Escritores
- Estudiantes
- Profesionales que necesitan tomar notas técnicas
