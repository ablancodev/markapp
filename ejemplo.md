# Visor de Markdown - Ejemplo

Bienvenido al **Visor de Markdown**. Este es un documento de ejemplo para probar todas las características.

## Características

- Modo claro y oscuro
- Exportación a PDF
- Edición en tiempo real
- Vista previa sincronizada

## Ejemplos de Formato

### Texto

Puedes escribir texto en *cursiva*, **negrita**, o ***ambos***.

También puedes usar ~~tachado~~.

### Listas

#### Lista no ordenada:

- Elemento 1
- Elemento 2
  - Subelemento 2.1
  - Subelemento 2.2
- Elemento 3

#### Lista ordenada:

1. Primer paso
2. Segundo paso
3. Tercer paso

### Código

Código en línea: `const x = 10;`

Bloque de código:

